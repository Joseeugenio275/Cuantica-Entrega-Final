# Cuantica-Entrega-Final
Aquí se incluyen los dos códigos en Julia para la Situación Problema de Física Cuántica. Incluye simulación de paquete gaussiano, efecto Hartman y análisis de resonancias.
El codigo de segunda entrega con adendo es el que trata sobre la entrega final, mientras que el primer codigo es para referenciar unas partes de la primera entrega en este entrega final.
